# Expire Bot

Бот для Matrix, который автоматически удаляет старые сообщения из комнат.

Добавляешь бота в комнату → даёшь права модератора → он удаляет сообщения старше заданного срока.

## Что умеет

- Автоматически удаляет старые сообщения (по умолчанию старше 24 часов)
- Работает в зашифрованных (E2E) комнатах
- Удаляет всё: текст, картинки, аудио, видео, файлы, стикеры, голосовые
- Не трогает системные события (название комнаты, участники, настройки)
- Настройки для каждой комнаты отдельно
- Переживает перезапуски — всё хранится в SQLite

## Быстрый старт

### 1. Создай аккаунт для бота на своём Matrix-сервере

**Если у тебя Synapse + MAS:**
```bash
# Создать пользователя
docker exec matrix-authentication-service \
  mas-cli manage register-user --yes expire-bot --password 'ПАРОЛЬ_БОТА'

# Задать пароль (нужен для работы в зашифрованных комнатах)
docker exec matrix-authentication-service \
  mas-cli -c /etc/mas/config.yaml manage set-password expire-bot 'ПАРОЛЬ_БОТА'
```

**Если у тебя Synapse без MAS:**
```bash
docker exec matrix-synapse \
  register_new_matrix_user -u expire-bot -p 'ПАРОЛЬ_БОТА' \
  -c /data/homeserver.yaml http://localhost:8008
```

### 2. Запусти бота

```bash
git clone https://github.com/USER/expire-bot.git
cd expire-bot
cp .env.example .env
```

Открой `.env` и заполни **три поля**:

```env
EXPIRE_BOT_HOMESERVER=https://matrix.example.com
EXPIRE_BOT_USER=@expire-bot:example.com
EXPIRE_BOT_PASSWORD=ПАРОЛЬ_БОТА
```

Запусти:
```bash
docker compose up -d
```

Готово. Бот работает.

### 3. Используй

1. Открой Matrix-клиент (Element и т.д.)
2. Пригласи бота в комнату — например `@expire-bot:example.com`
3. Бот зайдёт и пришлёт справку
4. Зайди в настройки комнаты → дай боту уровень прав **50** (модератор)
5. Всё — сообщения старше 24 часов будут удаляться автоматически

Хочешь другой срок? Напиши в комнату:
```
!expire set 7d
```

## Команды

Пишешь прямо в комнату:

| Команда | Что делает |
|---------|------------|
| `!expire set 1h` | Удалять сообщения старше 1 часа |
| `!expire set 7d` | Удалять сообщения старше 7 дней |
| `!expire 30m` | Короткая форма (то же что `set 30m`) |
| `!expire off` | Отключить автоудаление |
| `!expire status` | Показать текущие настройки |
| `!expire clean` | Удалить **всю** историю прямо сейчас |
| `!expire help` | Справка |

## Форматы времени

| Пример | Значение |
|--------|----------|
| `30m` | 30 минут |
| `1h` | 1 час |
| `6h` | 6 часов |
| `10h 15m` | 10 часов 15 минут |
| `1d` | 1 день |
| `1d 12h` | 1 день 12 часов |
| `7d` | 1 неделя |
| `30d` | 30 дней |
| `1y` | 1 год |
| `1y 6mo` | 1 год 6 месяцев |

Минимум — `1m` (1 минута), максимум — `365d` (1 год).

## Настройки

Все настройки задаются в `.env` файле. Обязательных всего три — остальные работают из коробки.

| Переменная | Обязательная | По умолчанию | Описание |
|------------|:------------:|:------------:|----------|
| `EXPIRE_BOT_HOMESERVER` | да | — | URL сервера (`https://matrix.example.com`) |
| `EXPIRE_BOT_USER` | да | — | ID бота (`@expire-bot:example.com`) |
| `EXPIRE_BOT_PASSWORD` | да | — | Пароль бота |
| `EXPIRE_BOT_DEFAULT_RETENTION` | нет | `24h` | Срок жизни сообщений для новых комнат |
| `EXPIRE_BOT_INTERVAL` | нет | `60` | Как часто проверять (в секундах) |
| `EXPIRE_BOT_MAX_REDACTS` | нет | `500` | Макс. удалений за один цикл |
| `EXPIRE_BOT_COMMAND_COOLDOWN` | нет | `5` | Пауза между командами от одного пользователя (сек) |
| `EXPIRE_BOT_PREFIX` | нет | `!expire` | Префикс команд |
| `EXPIRE_BOT_LOG_LEVEL` | нет | `INFO` | Уровень логов (`DEBUG`/`INFO`/`WARNING`/`ERROR`) |
| `EXPIRE_BOT_AVATAR` | нет | — | Путь к аватарке (`/data/avatar.png`) |
| `EXPIRE_BOT_NOTIFY_CLEANUP` | нет | `false` | Писать сколько сообщений удалено (`true`/`false`) |

Вместо `.env` можно использовать `config.yaml` — скопируй `config.yaml.example` в `data/config.yaml`.

## Аватарка

Положи картинку в `data/avatar.png` и добавь в `.env`:
```env
EXPIRE_BOT_AVATAR=/data/avatar.png
```

Аватарка загрузится один раз. Заменишь файл — обновится при следующем запуске.

## Права

- **Бот** должен иметь уровень прав 50+ (модератор) — иначе не сможет удалять
- Настраивать бота (`set`, `off`, `clean`) могут только модераторы комнаты (50+)
- `status` и `help` доступны всем
- Если у бота нет прав — он предупредит
